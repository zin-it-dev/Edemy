# Edemy 🎓

Discover and learn about any topic 🔖

📍 An online learning platform where you can join courses, track your personal learning journey, and share knowledge with the community 🐧

## 🧑‍💻 Technology Stack and Features

### 🖥 Backend (API)

- 🚀 **Django** with **Django REST Framework** for the Python backend API.
- 💾 **PostgreSQL** as the main SQL database.
- 🔑 **JWT token authentication** for secure sessions.
- 📘 **OpenAPI/Swagger** auto-generate API documentation.
- ✅ Tests with **Pytest**.
- 🔍 **Elasticsearch** for feature full-text searching.
- 🗽 **Celery** for async task and background jobs.
- 📚 **Redis** for caching memory.
- 📈 Chart.js for dashboards or data visualizations.

---

### 🌐 Frontend (Web)

- 💃 **React** with **TypeScript** and **Vite** for a modern frontend stack.
- 🎨 **React Bootstrap** for UI components.
- 🧪 **Playwright** for End-to-End testing.
- 🔋 **PWA** for website using application for web modern.

---

### 📱 Mobile (Hybrid)

- 📱 **React Native** for cross-platform mobile development (Android & iOS).
- 🧼 **Nativewind (Tailwind CSS)** for modern, utility-first styling.
- 🔑 **JWT token authentication** for secure sessions.
- 👑 **React Elements** for supprot UI Components base mobile.

---

### ⚙️ DevOps / Deployment

- 🧭 Shell automate setup and deployment.
- 🐋 **Docker Compose** for development and production.
- 📞 **NGINX** as reverse proxy / load balancer.
- 🚢 Deployment instructions using **Docker Compose**, including how to set up a frontend **NGINX** proxy to handle automatic HTTPS certificates.
- 🏭 **CI (continuous integration)** and **CD (continuous deployment)** based on **GitHub Actions**.

---

### 🧩 Third-Party Integrations

- 🔐 **Auth0** for authentication & authorization.
- ☁️ **Cloudinary** – media storage, image & video optimization.  
- 📫 **Email providers** (**MailTrap** for development, **SendGrid** for production).
